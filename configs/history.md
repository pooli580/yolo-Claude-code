## 常用参数参考

### 学习率推荐值
| 模型 | 初始 lr | 优化器 |
|------|---------|--------|
| YOLOv8n | 0.01 | SGD |
| YOLOv8s | 0.01 | SGD |
| YOLOv8m | 0.001 | SGD |
| YOLOv8l | 0.001 | SGD |
| YOLOv8x | 0.0005 | SGD |

### Batch Size 推荐
| GPU 显存 | 推荐 batch |
|----------|-----------|
| 8GB | 8-16 |
| 12GB | 16-24 |
| 16GB | 24-32 |
| 24GB | 32-64 |

### 问题诊断速查
| 现象 | 原因 | 解决方案 |
|------|------|----------|
| Loss 不下降 | lr 太小 | ×10 |
| Loss 震荡 | lr 太大 | ÷10 |
| 过拟合 | 模型过大 | 增加增强/正则化 |
| 欠拟合 | 模型太小 | 减小正则化/增大模型 |
